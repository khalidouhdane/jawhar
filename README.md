# Jawhar (جوهر) — Memorize with Meaning

[![Quran Foundation Hackathon 2026 Submission](https://img.shields.io/badge/Quran_Foundation-Hackathon_2026-black)](https://hackathon.quran.com)

The first Quran memorization companion built on understanding — where every verse you carry also carries its meaning. **Official submission for the 2026 Quran Foundation Hackathon.**

## What is Jawhar?

Jawhar helps you memorize the Quran while deeply understanding every verse — through translations, tafsir, reasons of revelation, and adaptive planning. Built with a minimalist B&W design where the Quran itself shines, not the interface.

### Three Pillars

1. **The Plan** — Adaptive daily plans using the traditional sabaq/sabqi/manzil methodology
2. **The Session** — Structured memorization sessions with timer, rep counter, and self-assessment
3. **The Understanding** — Translations, tafsir (brief + detailed), asbab al-nuzul, and surah introductions

## Platforms
