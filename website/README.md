# Jawhar Website

The marketing website for [Jawhar](https://github.com/khalidouhdane/jawhar) — *Memorize with Meaning*.

Built with [Next.js](https://nextjs.org) and deployed on [Vercel](https://vercel.com).

## Development

```bash
cd website
npm install
npm run dev
```

Opens at [http://localhost:3001](http://localhost:3001).

## Pages

- `/` — Landing page (Three Beats: Read, Understand, Memorize)
- `/download` — Platform downloads
- `/explore` — Interactive feature exploration
- `/privacy` — Privacy policy
- `/terms` — Terms of service
